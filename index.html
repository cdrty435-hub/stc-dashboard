📋 설치 순서 / ขั้นตอนการติดตั้ง
1
Google Sheets 열기
→ 확장 프로그램 → Apps Script
เปิด Sheets → ส่วนขยาย → Apps Script
2
기존 코드 전체 삭제 후
STC_apps_script.gs 코드 붙여넣기
ลบโค้ดเดิม วางโค้ดใหม่
3
SHEET_NAME 을 실제 시트 탭 이름으로 수정
예: "Sheet1" → "유지보수DB"
แก้ชื่อชีตให้ตรงกับชีตจริง
4
저장(Ctrl+S) → testAPI 함수 실행
→ 실행 로그에서 데이터 건수 확인
บันทึก → รัน testAPI ตรวจสอบ Log
5
배포 → 새 배포
유형: 웹 앱 | 실행: 나 | 액세스: 모든 사용자
→ URL 복사 후 위 입력창에 붙여넣기
배포 → เว็บแอป | คัดลอก URL วางด้านบน
